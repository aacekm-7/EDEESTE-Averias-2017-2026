# EDEESTE-Averias-2017-2026
Este conjunto de datos contiene las estadísticas de averías y emergencias atendidas de la Empresa Distribuidora de Electricidad del Este, S.A., (EDEESTE), generadas en el periodo 2017 - 2026, en el que se puede encontrar el detalle del reporte que el cliente hace cuando su servicio de energía eléctrica presenta algún problema técnico.
