Este proyecto utiliza el archivo “Estadisticas-de-Averias-y-Emergencias-Atendidas-por-Provincia-enero-2026.csv”, obtenido desde el portal oficial de datos abiertos de la República Dominicana (datos.gob.do). El flujo de trabajo consistió en cargar el CSV en Excel, realizar un proceso de limpieza, normalización y transformación de los datos, y posteriormente estructurarlos para su análisis en Power BI.

Durante la limpieza se corrigieron problemas de codificación en los textos, se estandarizaron nombres de columnas, se eliminaron inconsistencias y se preparó el dataset para un modelo tabular. Luego, en Power Query, se aplicaron transformaciones clave como la conversión de columnas mensuales a formato largo (unpivot), la creación de columnas de fecha, la generación de claves sustitutas (IDs) y la construcción de dimensiones como DimProvincia y DimTipoAveria. Finalmente, se integró todo en un modelo estrella dentro de Power BI para permitir análisis eficientes y visualizaciones dinámicas.

Este proyecto me permitió reforzar mis conocimientos en análisis de datos, Power Query, modelado dimensional, DAX, Excel avanzado y Power BI, aplicando buenas prácticas profesionales para la construcción de dashboards y modelos analíticos. También me ayudó a mejorar mi capacidad para trabajar con datos reales, detectar errores estructurales, transformar información en insights y documentar procesos de manera clara y reproducible.
